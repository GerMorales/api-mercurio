Para poder consumir la api, la pagina cuenta con tres endpoints: /comida, /cuadrimotos, /restaurante
https://api-mercurio.herokuapp.com/cuadrimotos
https://api-mercurio.herokuapp.com/comida
https://api-mercurio.herokuapp.com/restaurante

Tambien, es posible encontrar los elementos por id, utilizando los endpoints: /comida/id, /cuadrimotos/id, /restaurante/id
Por ejemplo
https://api-mercurio.herokuapp.com/cuadrimotos/1
https://api-mercurio.herokuapp.com/comida/3
https://api-mercurio.herokuapp.com/restaurante/2
